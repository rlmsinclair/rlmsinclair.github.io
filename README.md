# rlmsinclair.github.io

GitHub Pages site hosted at [www.trust.one](https://www.trust.one) — an interactive 3D visualisation of the DMCT (Decentralised Mutual Cascading Trust) protocol.

## What it shows

A full-screen WebGL animation depicting trust propagating as waves through spacetime — the visual centrepiece of the [DMCT project](https://github.com/rlmsinclair/dmct).

## Stack

Plain HTML/CSS/JavaScript — no build step, deploys directly via GitHub Pages.
